 Protein Tracker CLI:
Este é um projeto pessoal desenvolvido em Java com o objetivo de facilitar o controle diário de ingestão proteica para entusiastas de musculação e saúde. O projeto faz parte do meu portfólio de estudos como graduando em Engenharia de Computação no IF Goiano - Campus Trindade.

 Funcionalidades:
Cálculo Automatizado: Define a meta diária baseada no peso do usuário utilizando a métrica de 2g/kg.

Acompanhamento em Tempo Real: Loop interativo para adição de refeições e atualização do saldo restante.

Tratamento de Erros: Gestão de entradas inválidas (letras em vez de números) e cancelamentos de operação via blocos try-catch.

Interface Gráfica: Utilização da biblioteca javax.swing para interação via janelas de diálogo.

 Tecnologias Utilizadas
Linguagem: Java.

Interface: Swing (JOptionPane).

Controle de Versão: Git & GitHub.

Ambiente: IntelliJ IDEA.

Como Executar:
  Certifique-se de ter o JDK 17 ou superior instalado.

Clone o repositório:
  git clone https://github.com/natannn14/protein-counter.git.

Compile o arquivo:
  javac src/ContadorProteina.java.

Execute:
  java src/ContadorProteina.
